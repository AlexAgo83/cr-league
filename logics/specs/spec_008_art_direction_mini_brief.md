# Purpose
Define the initial visual direction for CR League.

# Direction
Competitive cartoon, slightly retro, readable, and not childish.

The game should feel like a modern digital board game about racing teams, not a serious Formula 1 simulator and not a chaotic mascot kart game.

# Visual Keywords
- top-down racing;
- bold team colors;
- clean track shapes;
- retro motorsport posters;
- compact garage dashboard;
- readable icons;
- tactile board-game clarity;
- light arcade energy.

# Avoid
- futuristic sci-fi racing;
- realistic F1 broadcast imitation;
- childish toy look;
- absurd slapstick tone;
- dark blurry stock imagery;
- overly glossy mobile casino feel;
- heavy purple/blue gradient identity;
- 3D complexity before gameplay proof.

# Cars
V1 cars:

- stylized top-down silhouettes;
- team color primary;
- small secondary accent;
- readable at small size;
- no licensed or real-world team lookalikes.

# Circuits
V1 circuits:

- simple readable top-down paths;
- distinct trait identity through shape and environment;
- event markers can appear around track;
- avoid high-detail backgrounds.

# UI
Main UI should feel like a team operations dashboard:

- next Grand Prix panel;
- preparation choices;
- card inventory;
- standings;
- race report;
- replay.

Use utility-first clarity. Do not make a marketing landing page as the first app screen.

# Palette Guidance
- Use multiple team colors.
- Keep UI neutral enough that team colors and weather states can stand out.
- Weather should be visually readable: dry, light rain, heavy rain.
- Avoid one-note beige, dark slate, or purple-blue dominance.

# Non-goals
- No final logo requirement.
- No full design system yet.
- No AI-generated asset dependency for V1.
- No 3D requirement.
- No photorealism.

# Open Questions
- What are 3 reference games or visuals to explicitly emulate?
- Should the app feel more like a sports dashboard or board game table?
- Should cars have characterful faces/details or stay more vehicle-like?
