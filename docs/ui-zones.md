# UI Zones

This file names layout zones that must stay stable across visual changes.

## Result / Replay

- `replay-copy-panel`: replay title and replay description.
- `replay-map-panel`: circuit heading, map, cars, playback controls, weather timeline, lap markers.
- `replay-moments-panel`: key moments list. Desktop: right of `replay-copy-panel`. Mobile: below `replay-map-panel`.

## Result / Report

- `report-phases`: race phase weather summary.
- `report-blocks`: rewards and key moments report content.
- `report-side-recap`: race recap side panel. Desktop: right column. Mobile: stacked below report content.
